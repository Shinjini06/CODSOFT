# 🤖 CodSoft AI Internship

Three AI projects built with Python and Streamlit.

---

## Task 1 — BotX Chatbot
A rule-based chatbot that answers questions about AI, Python, and math.

**Run:**
```bash
streamlit run task1_chatbot/chatbot.py
```

---

## Task 2 — Tic-Tac-Toe AI
Play Tic-Tac-Toe against an AI with Easy, Medium, and Hard difficulty levels using the Minimax algorithm.

**Run:**
```bash
streamlit run task2_tictactoe/tictactoe.py
```

---

## Task 3 — Image Captioning AI
Upload an image and get an instant AI-generated caption using color and brightness analysis.

**Run:**
```bash
streamlit run task3_imagecaption/imagecaption.py
```

---

## Setup
```bash
pip install streamlit Pillow requests
```
